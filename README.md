Mindful Intent Layer

AI-powered attention management app that reduces unconscious scrolling.

Problem

Students lose study time due to impulsive Instagram and YouTube opens.

Solution

Before opening distracting apps, users declare intent. The app predicts distraction patterns and creates mindful pauses.

Features

* Intent popup
* AI distraction insights
* Study mode
* Dashboard
* Focus streaks
* Friend leaderboard
